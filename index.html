Scripting API

Define any of these in your script. Anything you omit falls back to the GUI.

element_pattern(theta) → float

Axisymmetric element pattern. theta is measured from the element axis n̂ (which is tilted from +z by the Orientation slider). Returns a non-negative magnitude; the visualizer normalizes the peak to 1.

def element_pattern(theta):
    return abs(np.sin(theta))  # Hertzian dipole

excitations(N) → dict

Per-element amplitudes and phases. Return a dict with amps and phases (radians) of length N.

def excitations(N):
    n = np.arange(N)
    return {
        "amps":   np.cos(np.pi * (n - (N-1)/2) / (N-1) if N>1 else 0),
        "phases": np.zeros(N),
    }

pattern(theta, phi) → float

Full override. If defined, this replaces E·AF entirely. theta from +z, phi from +x. Used for cases that don't factor (mutual coupling, conformal arrays, etc.).

def pattern(theta, phi):
    return np.abs(np.cos(theta))**2

Available

np (numpy), math, print(...) → script console. The 3 functions above run once when you click Run; results are cached as sampled grids.

To go back to the GUI-driven element/amplitudes after running a script, click Revert to GUI.
